OM109-Signed Works

A financial system with no server room

Underneath the art sits a seven-layer, double-entry ledger running real Byzantine consensus across two nodes — the two phones, checking each other, so no single device is a single point of failure. It's the same architectural principle banks use, built without a bank's budget.

OM109 — signed twice, trusted once

Every certified work carries OM109, a dual-key signing algorithm built for this project: two independent cryptographic signatures combined into one fingerprint. Tamper with either signature alone, and the fingerprint breaks. It's not a gimmick — it's the same signer now certifying real documents for free through the Sovereign Registry.

Provenance you can hold in your hand — verified by math, not by asking you to trust our servers.

What "sovereign" actually means here

Every certificate this system issues is independently recomputable. The hash-chain math behind each record is public — anyone can check it without needing our servers to be online, without needing to trust us at all. That's the whole point of building it this way instead of the easy way.
